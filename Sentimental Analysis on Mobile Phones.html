Sentiment Analysis

A Master's level deep-dive into consumer emotion tracking across Twitter, Amazon, and YouTube using Python and Tableau.

Abstract

Sentiment Analysis (Opinion Mining) extracts insights from unstructured text data. This project utilizes text processing and ML techniques to collect reliable data from social platforms and provide actionable business insights regarding mobile phone features and brand perception.

Core Objectives

  • Analyze user responses to mobile features via social listening.
  • Identify emerging topics of interest in real-time.
  • Prioritize customer expectations for R&D teams.

The NLP Pipeline

End-to-End Architecture
